TAG

Overview

TAG is a performant Python/SQLite data processor that facilitates speedy analysis of tag-based data.

This demo version of TAG is designed for a public dataset of several million scraped Stack Overflow questions (availibe here--link)).

Input a list of tags (a query) and TAG will produce text and image files with pre-processed data ready for analysis.

TAG utilizes an inverted index (among other caches) to perform fast lookups and an algorithim based on TF-IDF to quickly find similar tags/queries.

See the public GitHub repository for help setting up and running the program (link).

See how the query stacks up against frequently-referenced tags with charts!
The scatterplot helps visualize the posting frequency of the tag, how those posts are ranked, and any relation between the word count of a post and it's score.









Similar Tags

Using an algorithim similar to the Term Frequency - Inverse Document Frequency (TF-IDF) algorithm, TAG will rank tags based on similarity to the query. This is helpful to find tags related to the query. The two columns in the analyzed data help differentiate if a tag was ranked similar because it is very popular across the dataset (TF) versus tags that occupy a niche that overlaps with the query (IDF).
